# [논문 요약] Fully Convolutional Networks (FCN) for Semantic Segmentation

## 1. 핵심 요약
- 이 논문은 CNN 단독으로 pixels-to-pixels 방식의 **end-to-end 학습**을 진행했을 때, 의미론적 분할(Semantic Segmentation) 분야에서 기존의 최고 성능(state-of-the-art)을 뛰어넘는다는 것을 보여준다.
  - **end-to-end 방식:** `입력 = 원본 이미지 데이터`, `출력 = 원하는 최종 결과물(정답)`인 방식. 중간에 사람이 수동으로 특징을 뽑아주거나 별도의 알고리즘을 끼워 넣을 필요 없이, 거대한 신경망 하나에 입력값과 출력값(정답)만 주면 네트워크 스스로 수많은 가중치를 조절해가며 규칙을 찾아내는 방식이다.
  - 이 논문에서 표현한 **“trained end-to-end, pixels-to-pixels”**란, 모델에 고양이 사진을 넣으면(pixels) 복잡한 중간 처리 없이 모델이 알아서 픽셀 단위로 색칠된 분할 결과(pixels)를 출력해내는 것을 의미한다. 결과물이 정답과 다르면 Loss를 계산해 역전파하며 모델 전체를 통째로 학습시킨다.
- **핵심 아이디어:** 임의의 크기를 가진 입력을 받아 동일한 크기의 출력을 만들어내면서도 효율적으로 추론과 학습을 수행하는 **완전 합성곱 신경망(Fully Convolutional Networks)**을 구축하는 것.
- 완전 합성곱 신경망의 영역을 정의 및 구체화하고, 이를 공간적으로 밀집된 예측(spatially dense prediction)에 어떻게 적용하는지 설명한다.
- 최신 분류 신경망들(AlexNet, VGG net, GoogLeNet)을 완전 합성곱 신경망으로 변형하고, **미세 조정(fine-tuning)**을 통해 이 신경망들이 학습한 표현들을 분할(Segmentation) 작업에 전이시킨다.
  - **미세 조정:** 이미 방대한 데이터로 학습된 모델을 가져와 목적에 맞게 추가 학습시키는 전이 학습(Transfer Learning)의 일종.
  - 분류 대회 우승 모델들의 마지막 클래스 분류 층을 잘라내고, 픽셀 단위 예측이 가능하도록 완전 합성곱 형태로 구조를 변경한 뒤 분할용 정답 데이터를 주며 추가 학습을 진행했다.
- 깊고 거친(coarse) 계층의 의미론적 정보와 세밀한 계층의 형태(appearance) 정보를 결합하여 정확하고 디테일한 분할 결과를 만들어내는 **스킵 아키텍처(skip architecture)**를 정의한다.
  - **스킵 아키텍처:** 이미지가 모델의 깊은 층을 통과하면 전체적인 맥락은 파악하지만, 압축으로 인해 세밀한 위치 정보는 잃게 된다. 이를 원본 크기로 키우면 윤곽선이 뭉개지므로, 네트워크 중간 단계를 건너뛴 원본을 최종 결과물에 섞어주는(Fusion) 방식이다.

## 2. 아키텍처 (Architecture)

- ILSVRC 분류기들을 완전 합성곱 신경망(FCN)으로 변환하고, 네트워크 내부 업샘플링(in-network upsampling)과 픽셀 단위 손실(pixelwise loss)을 추가하여 밀집 예측(dense prediction)이 가능하도록 모델을 증강시켰다.
- fine-tuning을 통해 분할을 학습시킨다.
- 거칠지만 의미론적인 정보와 국소적이고 형태적인 정보를 융합하기 위해 계층 간에 스킵을 추가한다.
- 이 스킵 아키텍처는 출력물의 의미와 공간적 정밀도를 정제하기 위해 엔드투엔드로 학습된다.

### 2.1 실험 환경 설정
- **데이터셋:** PASCAL VOC 2011 분할 챌린지 데이터셋 사용
- **손실 함수:** 픽셀별 다항 로지스틱 손실 사용
- **평가 지표:** 배경을 포함한 모든 클래스에 대한 평균 픽셀 교차율(mean IU, Intersection over Union) 사용
- **학습 조건:** 정답(ground truth) 데이터에서 모호하거나 어렵다고 판단되어 마스킹 처리된 픽셀들은 무시함

### 2.2 분류기에서 밀집 예측 FCN으로 (4-1)
- **기반 모델 선정:** 19-layer 신경망과 성능이 동일한 **VGG 16-layer** 신경망 선택. (GoogLeNet의 경우 최종 손실 계층만 사용하고 마지막 평균 풀링 계층은 버려 성능 향상)
- **세부 개조 작업:**
  - 최종 분류기 계층(classifier layer) 절단(decapitate).
  - 모든 완전 연결 계층(FC layers)을 합성곱으로 변환.
  - PASCAL 클래스 점수 예측을 위해 채널 차원이 21인 1x1 합성곱 계층 추가.
  - 거친 출력들을 픽셀 단위로 조밀하게 만들기 위해 이중 선형 업샘플링(bilinearly upsample)하는 디컨볼루션(deconvolution) 계층 추가.
- **초기 실험 결과:** FCN-VGG-16은 검증 세트에서 평균 IU 56.0을 기록(당시 테스트 셋 최고 기록 52.6)하며 이미 최고 수준의 성능을 입증. 추가 데이터 학습 시 59.4까지 성능 상승.

### 2.3 What and Where의 결합 (4-2)
- **도입 배경:** 완전 합성곱 형태로 변환된 분류기들의 출력물은 불만족스러울 정도로 거칠다(coarse). 최종 예측 계층에 존재하는 32픽셀 스트라이드(stride)가 출력물의 디테일을 제한하기 때문.
- **해결책 (스킵 추가):** 최종 예측 계층을 더 조밀한(finer) 스트라이드를 가진 하위 계층과 결합. 세밀한 계층과 거친 계층을 결합하면 전역적인 구조를 유지하면서도 지역적인 예측이 가능해진다.
- **FCN-16s 설계 및 학습:**
  - 16 픽셀 스트라이드 계층에서 예측을 수행. pool4 위에 1x1 합성곱 계층 추가.
  - 2배 업샘플링 계층을 추가하고, 이 결과를 conv7 위에서 계산된 예측값과 융합(fuse).
  - 결과: 검증 세트 성능이 평균 IU 기준 3.0 향상된 63.4 기록.
- **FCN-8s 설계 및 한계:**
  - pool4 및 conv7의 융합 예측값을 2배 업샘플링한 결과에 pool3의 예측값을 융합.
  - 결과: 62.7 평균 IU로 소폭 개선되었으나 수확 체감(diminishing returns)에 직면하여 더 하위 계층 융합은 중단.
- **실패한 방법들:** 풀링 계층의 스트라이드를 줄이는 직관적인 방법은 VGG16 기반에서 계산 비용 폭증 및 학습 난조를 일으킴.

### 2.4 실험 프레임워크 (4-3)
- **최적화 기법 (Optimization):**
  - 모멘텀(0.9)을 적용한 확률적 경사 하강법(SGD) 사용. 미니 배치 크기 20.
  - 고정 학습률: FCN-AlexNet($10^{-3}$), FCN-VGG16($10^{-4}$), FCN-GoogLeNet($5^{-5}$).
  - 학습은 '학습률' 자체에 가장 민감하게 반응함.
- **Fine-tuning:** 출력 분류기만 학습하면 성능의 70% 수준. 신경망 전체를 역전파로 fine-tuning.
- **추가 학습 데이터:** PASCAL VOC 2011 (1,112장) 외에 SDS 학습용 8,498장 라벨을 추가 수집하여 사용. 성능 향상(평균 IU 59.4).
- **패치 샘플링의 실효성:** 이미지를 잘게 쪼개는 것보다, 전체 이미지를 넣고 겹치는 연산을 한 번에 처리한 뒤 필요 없는 결과값을 솎아내는 것이 연산 속도에 유리하여 **전체 이미지 학습** 채택.
- **클래스 불균형 처리:** 라벨 불균형(배경 3/4)이 있으나, 완전 합성곱 학습의 특성상 별도의 클래스 균형 작업 불필요.
- **데이터 증강:** 좌우 반전 및 지터링(이동)을 통한 데이터 증강을 시도했으나 성능 향상은 미미했음.

## 3. 실험 결과 (Metrics)

의미론적 분할 및 장면 구문 분석(scene parsing) 평가에서 흔히 쓰이는 픽셀 정확도(pixel accuracy)와 영역 교차율(IU)의 변형인 4가지 지표를 사용.

$$
\text{Pixel Accuracy} = \frac{\sum_{i} n_{ii}}{\sum_{i} t_{i}}
$$

$$
\text{Mean Accuracy} = \frac{1}{n_{cl}} \sum_{i} \frac{n_{ii}}{t_{i}}
$$

$$
\text{Mean IU} = \frac{1}{n_{cl}} \sum_{i} \frac{n_{ii}}{t_{i} + \sum_{j} n_{ji} - n_{ii}}
$$

$$
\text{Frequency Weighted IU} = \left(\sum_{k} t_{k}\right)^{-1} \sum_{i} \frac{t_{i} n_{ii}}{t_{i} + \sum_{j} n_{ji} - n_{ii}}
$$

- $n_{cl}$ : 전체 클래스(Class)의 개수 (예: 배경, 고양이, 강아지 = 3개)
- $t_{i}$ : 실제 정답 사진에 있는 $i$ 클래스의 전체 픽셀 수 (정답 면적)
- $n_{ij}$ : 실제 정답은 $i$ 인데, 모델이 $j$ 라고 예측한 픽셀 수

---

## 4. 결론

FCN은 현대의 이미지 분류용 합성곱 신경망들의 기반이 되었다. 이러한 분류 신경망을 분할 작업으로 확장하며, 다중 해상도 계층 조합(스킵 아키텍처)을 통해 구조를 개선하여 기존의 최고 성능을 극적으로 향상시켰다. 또한 학습과 추론의 과정을 단순화하고 속도를 대폭 높였다.
